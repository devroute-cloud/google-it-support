## Topic
File System Structure (Windows & Linux)

## What was studied
- Differences between Windows and Linux file systems
- Directory hierarchy in both operating systems

## What was practiced
- Navigating directories
- Identifying file paths
- Exploring system structure

## Operating Systems
- Linux (Ubuntu)
- Windows

## Commands Used
### Linux
- pwd → show current directory
- ls → list files and directories
- cd → navigate between directories

### Windows
- dir → list files and directories
- cd → change directory
