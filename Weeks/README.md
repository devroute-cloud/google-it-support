# IT Support Learning Topics

## 📄 Fundamentals
- File system structure (Windows & Linux)
- CLI navigation (`cd`, `ls`, `dir`)
- File management (create, move, delete)
- File search (`find`, `dir`, `where`)
- Basic text manipulation

---

## ⚙️ Core Commands
- Linux: `ls`, `cd`, `pwd`, `mkdir`, `rm.`
- Windows CLI: `dir`, `cd`, `copy`, `del.`
- Permissions: `chmod`, `chown.`
- Linux users: `useradd`, `passwd`, `userdel.`
- Windows users: `net user.`

---

## 🔐 Users & Permissions
- User and group management (Linux & Windows)
- File and folder permissions
- Roles: user, group, root/admin

---

## 📦 Software & System
- Linux packages: `apt`, `yum.`
- Windows software installation
- System updates (Linux)
- File compression: `zip`, `tar`, `gzip.`
