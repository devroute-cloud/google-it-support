
### Navigation
- Navigation is the process of moving through your computer's folder structure (the File System). Since you don't have a visual window open, you use commands to "travel" from one folder to another.

**Topics**
- Absolute vs relative paths: An Absolute path is the complete address from the very root of the system (e.g., /home/user/docs). A Relative path is a shortcut based on where you are right now (e.g., Documents/Project).
- Current working directory: This is your "current location"—the specific folder you are currently "inside" in the terminal.
- Changing directories: The act of moving your "location" from one folder to another.
- Listing files and folders: Peeking inside your current folder to see what’s there.

**Commands**

Linux:
- `pwd` = Print Working Directory: Tells you exactly where you are so you don't get lost.
- `ls` = List: Shows you all files and subfolders in your current spot.
- `cd` = Change Directory: This is your "vehicle" to move into different folders.

Windows:
- `dir` = Directory: The Windows version of ls; it lists everything in the folder.
- `cd` = Change Directory: This is your "vehicle" to move into different folders.
