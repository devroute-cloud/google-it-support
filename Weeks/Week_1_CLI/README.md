# Week 1 – CLI & File Management

This week focuses on building core command line and file management skills in both Linux and Windows environments.

The objective is to understand how to navigate file systems, manage files and directories, search for data, and perform basic text operations using the command line.

## Topics Covered

- File system navigation
- Absolute and relative paths
- File and directory creation
- Copy, move, and delete operations
- Search for files and text
- Basic text viewing and editing

## Operating Systems Used

- Linux
- Windows

## Commands Practiced

### Linux
- `pwd`
- `ls`
- `cd`
- `mkdir`
- `cp`
- `mv`
- `rm`
- `find`
- `grep`
- `cat`
- `head`
- `tail`

### Windows
- `dir`
- `cd`
- `mkdir`
- `copy`
- `move`
- `del`
- `findstr`

## Tools Used

- Bash
- Windows Command Prompt
- File System Utilities

## Goal

Develop confidence using the command line and performing common file management tasks required in IT Support roles.
